**6 sikkerhetsscenarier:**
1. **Prevents Privilege Escalation**
   - Attack: User modifies role to gain admin access
   - Defense: All role changes audited, deny-by-default prevents escalation

2. **Stops Broken Access Control**
   - Attack: Direct object reference bypass (IDOR)
   - Defense: Resource-level checks verify ownership, attributes validate access

3. **Detects Unauthorized Access**
   - Attack: Brute-force permission probing
   - Defense: Rate limiting, anomaly detection, audit logs detect and prevent probing

4. **Zero-Trust Every Request**
   - Attack: Token reuse or session hijacking
   - Defense: Principal verification on every request, no token reuse, session validation

5. **Policy Injection Protection**
   - Attack: Policy injection or manipulation
   - Defense: Compiled policies, input validation, version control prevent injection

6. **Defense in Depth**
   - Attack: Multi-vector coordinated attack
   - Defense: Multiple security layers, one breach doesn't compromise system
